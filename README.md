# 🚀 Quality Assurance & Test Automation Portfolio

This repository showcases an end-to-end QA testing suite built with **Playwright**, **JavaScript**, **Postman**, and **SQL**. It includes UI automation using the Page Object Model (POM), direct API verification, and relational database testing.

---

## 🛠️ Tech Stack & Skills
* **Automation Framework:** Playwright (JavaScript)
* **Design Pattern:** Page Object Model (POM)
* **API Testing:** Postman & Playwright `request` API context
* **Database Querying:** SQL (INNER JOINs, data integrity checks)
* **CI/CD:** GitHub Actions

---

## 📁 Repository Structure
* `/pages` - Modular Page Object Model classes (`LoginPage.js`, `SearchPage.js`)
* `/tests/ui` - End-to-end browser automation scripts
* `/tests/api` - Direct API endpoint verification tests
